# 👑 핵심 서비스
## 3가지 전기차 추천 플로우
### 🚩 현재 차량을 등록하여 등록한 차량의 브랜드와 모델을 인공지능 모델로 예측합니다
- 회원가입 시, 현재 차량 소지 여부를 확인 후, 차량이 없다면 본인이 원하는 드림카를 업로드 하도록 유도합니다.
<br/>

### 🚩 현재 보유 중인 차량의 실제 연비를 계산하여 추천합니다
- 차량을 소지한 경우 주유 내역을 통해 연비를 계산 후, 그에 해당하는 전비를 가진 전기차를 추천해드립니다.
